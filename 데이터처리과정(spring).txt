웹의 데이터 통신 과정은 다음과 같다.
클라이언트 요청 -> 이벤트 발생 -> 데이터를 컨트롤러에 전달(비동기 통신) -> 서비스단과 DAO를 통하여 데이터 가공 및 처리 -> 마지막으로 컨트롤러
반환 값을 js 비동기 통신에 전달 -> 다시 js 에서 추가 필요한 작업 처리 -> 클라이언트에 반영

이벤트 처리 과정 순서

js파일에서 등록된 이벤트 리스너를 통해 클라이언트에서 요청한 정보를 담아 보냄
-> 서버에서는 request 객체, response객체와 클라이언트에서 보낸 데이터를 받아 컨트롤러 파라미터로 정의된 빈 객체에 주입해준다.
-> 그뒤에 서비스 함수를 호출 -> DAO를 통해 DB처리 -> 반환값 다시 비동기 통신으로 전달 ->다시 js 에서 추가 필요한 작업 처리
 -> 클라이언트에 반영

*BaseMultiActionController.getResponseData()의 세번째 파라미터는 비동기 통신에서 변수 data
