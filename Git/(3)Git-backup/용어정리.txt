용어 정리

Local Repository(지역 저장소) : 실제 작업하여 버전생성하는 컴퓨터
Remote Repository(원격 저장소) : Local Repository 의 작업내용을 업로드 하는곳

Push : 지역 저장소의 자원을 원격 저장소로 업로드(백업) 하는 과정
Clone : 원격 저장소의 자원을 main 지역 저장소가 아닌 제2지역 저장소에 복제 하는 과정
Pull : 아직 main 지역 저장소에는 없지만 제2지역 저장소에서 갱신한 내용을 원격 저장소에 Push하여
제2지역 저장소와 똑같이 갱신된 원격 저장소를 통해 main지역 저장소에 다운로드 받는 과정

Git hosting : 원격 저장소의 역할을 인터넷을 통해 원격으로 서버를 제공해 주는 것

*통신방법 
  http : 쉽고 간단하지만 보안이 약하다 , ssh : 복잡하지만 보안에 강하다
