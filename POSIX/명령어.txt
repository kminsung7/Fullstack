명령어 1

pwd : 현재 위치 경로
/ : 루트 디렉토리
~ : 홈 디렉토리 (터미널을 켤때 기본으로 설정된 디렉토리)
cd : 디렉토리 이동 명령어
ls : 현재 디렉토리의 파일과 폴더를 보여줌
*명령어 상세 설명 보는 법 : man '명령어' 입력
*명령어 뒤 --help : 간단한 메뉴얼
*q버튼을 누르면 빠져나올 수 있다
*ls -l을 수행하고 나오는 목록에 가장 첫번째 글자가 d이면 폴더라는 뜻이고 -이면 파일이라는 뜻임
touch : 의미없는 텍스트 문서 생성
*posix 계열의 시스템들은 파일명 앞의 '.'은 숨김파일을 의미한다
mkdir : 폴더 생성
./ : '/'앞의 '.'은 현재 디렉토리를 의미한다
../ : '/'앞의 '..'은 상위(이전) 디렉토리를 의미한다
mv : 폴더,파일 이동 및 이름 변경
mv dummy dummy2 : dummy 폴더의 이름을 dummy2로 바꿈
rm dummy : dummy 파일 및 폴더 삭제 *폴더 삭제는 위험함으로 실제 삭제를 위해서는 rm -r dummy라고 입력해야한다
