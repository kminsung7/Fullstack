명령어 2 

절대경로 : 현재 어떠한 경로에 있던 절대경로를 입력하면 그곳을 가리킨다.
상대경로 : 현재 위치에 따라 경로 명령어에 의한 위치가 다르거나 원하는 경로로 가는 경로 명령어의 값이 달라진다
ex) /Users/live/posix 의 디렉토리를 구성하는 곳에서
만약 현재 디렉토리(경로)가 /Users/live 일때 posix 디렉토리로 가기 위한 방법은
절대경로 : /Users/live/posix ,상대경로 : ./posix 라고 입력하면 된다 *상대경로의 ./는 생략 가능
* nano : 텍스트 편집 프로그램 *nano 편집기 안에 아래의 명령어 가이드의 ^표시는 Ctrl 이다
cat : 파일안의 텍스트릴 출력해줌
* 터미널 창에서 명령어 입력중 도중에 취소 하고 싶으면 Ctrl + c 를 입력한다
mv hello.txt hello_1.txt : 파일 이름 변경
mv hello.txt ../hello.txt : 상위 폴더로 이동
; : 하나의 명령어가 끝난뒤 ';'를 붙여주고 뒤에 이어 다른 명령어를 입력하고 또';'를 입력해서 명령어를 연속적으로 입력하게 할 수 있다. 즉, ';'는 명령어 구분자라고 할 수 있다
명령어 뒤 '&&' : 단순 ';'를 사용하면 명령어를 잘 못 입력해도 다음 명령어가 수행된다 이는 심각한 오류를 발생할 수 있다
때문에 '&&' 를 사용하는데 이는 앞의 명령어가 성공적으로 수행되면 다음명령어의 수행으로 넘어가 줄 수 있게 해준다
